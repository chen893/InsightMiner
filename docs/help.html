常见问题

1. 点击首页的"开始分析"按钮或导航栏的"需求分析"

2. 在文本输入框中输入您要分析的内容

3. 点击"开始分析"按钮，等待分析结果

4. 查看分析结果，可以导出或保存到历史记录

目前支持以下文件格式：

  • 文本文件 (.txt)
  • Word文档 (.docx)
  • PDF文件 (.pdf)

在分析结果页面，您可以：

  • 点击"导出PDF报告"按钮导出完整报告
  • 点击"导出Excel"按钮导出数据表格
  • 使用"分享结果"功能生成分享链接

在历史记录页面，您可以：

  • 查看所有历史分析记录
  • 使用搜索和筛选功能找到特定记录
  • 编辑或删除单条记录
  • 清空所有历史记录
